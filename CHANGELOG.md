# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-07

### Added

Initial release of Contexxt

- Context menu "Inspect with Contexxt" to view element details.
- Image and Video inspection (dimensions, source, size).
- Text/Element inspection (font, color, classes).
- Direct download and Open in New Tab actions.
- Ctrl+Click shortcut.
